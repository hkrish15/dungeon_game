# dungeon_game
DUNGEON CRAWLER
 A simple 2D dungeon crawler game where the player navigates a character through a maze-like dungeon, avoids enemies, collects items, and exits the level.

Core Features:
Player Movement:
Move the character using arrow keys or WASD.

Dungeon Map:
Grid-based system for the dungeon layout is used.
Include walls, pathways, and an exit are included.

Enemies:
Simple AI-controlled enemies that move randomly or chase the player.

Collectibles:
Items like coins or keys for the player to collect.
Collected items and score on the screen is displayed

Game Over/Win Conditions:
Game Over if the player collides with an enemy.
Player wins upon reaching the exit.
